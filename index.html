About

I work on gravitational-wave and black-hole physics, focusing on extracting weak signals from noisy experimental data and searching for ideal high frequency candidate events. My research blends theoretical modeling with practical signal-processing pipelines, including phase-coherent analysis, matched filtering, and robust PSD estimation for resonant microwave cavity systems.

Research software and methods

Signal Processing in Resonant Microwave Cavitites

Phase-coherent analysis pipelines developed in collaboration with ADMX scientists, targeting MHz–GHz gravitational-wave searches using resonant microwave cavity detectors.

  • FFT binning with phase preserved, receiver-shape flattening, and precise frequency-axis bookkeeping
  • Welch PSD estimation and robust stacking across many high-resolution cavity scans
  • Frequency-domain matched filtering using physically consistent PSD models for narrowband GW signals
  • Code and analysis tools available at github.com/cewasiuk

Modeling and analysis

Theory-to-code pipelines for black-hole evolution and gravitational-wave phenomenology.

  • Numerical pipelines: ODE evolution, interpolation/resampling, parameter scans
  • Statistical inference and noise modeling for detector sensitivity estimation
  • Machine learning experience: CNN / U-Net tumor segmentation and severity grading (Erdős Institute capstone)

Technical Expertise

Signal Processing & Data Analysis

I specialize in extracting weak, narrowband signals from noisy experimental data, with an emphasis on phase-coherent analysis for resonant detector systems.

  • Matched filtering and optimal detection statistics
  • Power spectral density estimation (Welch, median and robust stacking)
  • Fourier-domain analysis (FFT / rFFT), frequency binning, and resampling
  • Noise modeling, stationarity checks, and statistical inference
  • Time–frequency analysis for long-duration and quasi-monochromatic signals

Computational Modeling & Scientific Computing

I build end-to-end numerical pipelines that connect theoretical models directly to experimental data products, emphasizing reproducibility, performance, and clarity.

  • High-performance numerical simulations and parameter scans
  • ODE-based evolution models and numerical integration
  • Monte Carlo methods and uncertainty propagation
  • Interpolation, regridding, and large-scale data aggregation
  • Structured data formats (HDF5) and metadata-aware workflows

Machine Learning & Data Science

Experience applying modern machine-learning techniques to real-world scientific and medical datasets, with a focus on interpretability and uncertainty awareness.

  • Deep learning with CNN and U-Net architectures
  • Medical image segmentation and severity classification from MRI data
  • Model evaluation using Dice, IoU, ROC, and calibration metrics
  • Uncertainty quantification and robustness analysis
  • End-to-end ML pipelines: preprocessing → training → validation → inference
