Projects


M.E.L.T Project Research Proposal

A soon to be published paper oriented at the prospect of using laser sintering to mitigate the harmful effects of lunar regolith on lunar technology and astronauts.

Measuring the Hubble Constant Using Sodium D Redshift and Graviational Waves

This paper outlines the analysis of given data to determine the Hubble Constant. This was done using the noticable redshift to calculate the recessonal velocity of a far away galaxy. We were then able to process continuous signal data from gravitational waves to calculate the proper distance to the galaxy.

Electronic Survival Past The Lunar Night

A second soon to be published paper detailing the numerous problems we faced when attempting to create a lunar resilient laser. This paper outlines the possible designs that could be incorporated to improve longevity of technology on the lunar surface.

Coriolis Force on a Two Dimensional Object Due to the Earth's Rotation

A summary of a coding project designed to illustrate the deviation of an object from a linear path due to the effect of an external Coriolis Force

Measuring the Index of Refraction Using a Michelson Interferometer

Using a Michelson Interferometer, I was able to calculate the index of refraction of air by slowly releasing air into a vacuum chamber. By measuring the change in number of interference patterns on a screen, I was able to measure an index value that was both accurate and precise compared to the actual value.

Measuring the Astronomical Unit Using Ceres at Opposition

Using the transit of the dwarf planet ceres, I was able to collect data over multiple days to calculate the Astronomical Unit. This was done by subtracting the rotation of the Earth to linearize our data, and then using parallax angles to calculate the distance from the Earth to the Sun.
